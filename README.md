# global-caijing-guide

本仓库用于归档和发布多个独立 HTML 页面。这些页面以自包含的静态文件形式存在，不依赖外部服务器或特定域名。

## 项目简介

本项目旨在整理和保存一系列独立的 HTML 文档。每个页面均为完整的、可独立运行的网页文件，便于离线浏览或分发。仓库不针对任何特定网站或域名，仅作为文件归档与展示的载体。

## 目录结构

仓库根目录下直接存放 HTML 文件，可根据需要创建子目录进行分类。典型结构如下：

```
global-caijing-guide/
├── index.html          (可选入口页面)
├── page-1.html
├── page-2.html
├── subdir/
│   ├── example.html
│   └── ...
└── README.md
```

## 页面归档说明

- 所有页面均为纯 HTML 文件，无需后端支持。
- 文件命名建议使用英文小写字母、数字和连字符，避免特殊字符。
- 每个页面应尽量自包含，CSS 和 JavaScript 建议内联或使用相对路径引用同一仓库内的资源。
- 页面内容不指向任何具体商业、新闻或营销网站，仅作为技术演示或资料存档用途。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或改进现有页面。
- 新增页面请在 README 中简要说明其用途（可选）。
- 请确保文件不包含恶意代码、跟踪脚本或违反 GitHub 服务条款的内容。
- 本仓库不保证定期更新，但会尽力保持文件可访问。
